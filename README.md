# Todo App

This is a simple Todo application built with React and Firebase.  
It supports user authentication, category management, and todo items CRUD operations.

## Features

- User authentication
- Create, edit, and delete categories
- Add, edit, delete, and toggle todo items within categories
- Real-time synchronization with Firebase Firestore
- Responsive UI with light/dark theme support

## Live Demo

You can check out the live app here:  
[https://plantri-da833.web.app/](https://plantri-da833.web.app/)

## Installation and Running Locally

1. Clone the repository:
   git clone git@github.com:MilanVujevic/plantri.git

2. Install dependencies:
    npm install

3. Setup your Firebase config in the firebase.js file.

4. Start the app:
    npm start

## Technologies Used

- React
- Firebase Authentication
- Firebase Firestore
- Tailwind CSS
